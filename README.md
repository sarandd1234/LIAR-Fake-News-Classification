# LIAR-Fake-News-Classification

## Project Overview
This project uses the LIAR dataset to classify political statements based on truthfulness. The goal was to apply natural language processing and supervised machine learning to analyze patterns in misinformation.

## Tools Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-Learn
- NLTK
- Matplotlib
- LIME

## Methods
The project included data cleaning, text preprocessing, feature transformation, model training, and performance evaluation. A Random Forest classifier was used to predict the credibility of political statements.

## Key Steps
1. Loaded and explored the LIAR dataset
2. Cleaned and preprocessed text data
3. Transformed text into numerical features
4. Trained a Random Forest classification model
5. Evaluated model performance using accuracy, precision, recall, and F1-score
6. Used LIME to explain model predictions

## Results
The model was evaluated using standard classification metrics and visualizations, including a confusion matrix and classification report.

## What I Learned
This project strengthened my skills in machine learning, text preprocessing, model evaluation, and explainable AI. It also showed how challenging misinformation classification can be because political language is often complex and context-dependent.
